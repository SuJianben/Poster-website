# PosterStore Shopify Theme

An Online Store 2.0 theme converted from the supplied Figma-generated homepage.

## Shopify setup

1. Upload this folder as a theme in **Online Store → Themes**.
2. In the theme editor, set the hero and editorial images, category links, and featured collections.
3. Create products and collections; all product displays and cart actions use Shopify's native data and checkout.

The source React/Figma project remains in the sibling `海报跨境独立站Demo` directory and is not part of the Shopify theme.
